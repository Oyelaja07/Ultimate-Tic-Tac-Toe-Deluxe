# Changelog

## Version 2.0.0

- Added one-player and two-player modes.
- Added Easy, Medium, and Hard AI.
- Added player names and symbol selection.
- Added match targets of 3, 5, and 10 wins.
- Added automatic next-round reset without losing the score.
- Added themes, sounds, match history, undo, and keyboard controls.
- Added GitHub-ready documentation and project files.
